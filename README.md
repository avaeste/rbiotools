# RBioTools

**RBioTools** is an R package for **comparative microbial genomics**.  
It is designed to be used in **RStudio** and works with **GenBank (NCBI) IDs** to perform analyses and generate figures.

---

## Installation

### Requirements

- **R** (https://www.r-project.org)
- **RStudio Desktop** (https://posit.co/download/rstudio-desktop)

#### System tools (required to compile packages)

- **Windows**: Install **Rtools42**  
  https://cran.r-project.org/bin/windows/Rtools/rtools42/rtools.html

- **macOS**: Install Xcode Command Line Tools
